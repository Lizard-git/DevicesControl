Обновления
v.2.0.0
  1. Обновленно:
    1. Полностью переработан внешний вид
    2. Улучшено взаимодействие с программой с учетом рекомендаций пользователей
    3. Авторизация:
      1. Логины при входе в программу теперь можно не запоминать, а искать и выбирать их.
      2. Логином будет являться имя доменной учетной записи
      3. Cессия в программе будет длиннее. (Реже теперь будет необходимо вводить логин и пароль)
    4. Главная страница:
      1. Cтраница более не является функциональной, теперь она информационная.
      2. Теперь на главной странице указана небольшая информация о сотруднике
    5. Взаимодействие с устройствами:
      1. Добавленна сортировка для каждого столбца таблиц
      2. Поиск теперь работает по всем параметрам таблиц (с помощью него можно теперь сортировать данные и выгружать их в Excel)
      3. Теперь производители устройств будут едины. Их придется выбирать, а не писать самостоятельно (Не будет того, что кто-то написал Aquarius, а кто-то AQUARIUS ну и самое забавное Aguarius)
      4. Cтатусы перерабоаны. Теперь в программе имеются все необходимые для этого статусы.
      5. Убран промежуточный этап просмотра информации об устройстве. Теперь редактирование доступно сразу.
      6. В описании устройства убрано поле "Ответсвенный" просто за ненадобностью.
    6. Работа с адресами:
      1. Переработана логика адресов (зарегистрированных зданий). Все адреса теперь имеют район, населенный пункт, улицу и дом. Без всяких сокращенных названий
    7. Работа с пользователями:
      1. Добавлена возможность редактирования пользователей (как для администраторам, так и самим пользователям(с ограничениями))
      2. Добавлена возможность удаления пользователей (только администраторам)
      3. Все пользователи могут просматривать кто зарегистрированн в программе, но изменять могут только администраторы
      4. Cессия в программе будет длиннее. (Реже теперь будет необходимо вводить логин и пароль)
    8. Другое:
      1. У программы появилась версионность. Версия указанна на страница авторизации и на вкладке "Информация".
      2. Добавлена страница "Информация". На ней можно будет узнать о текущей версии программы и о списках изменений.
